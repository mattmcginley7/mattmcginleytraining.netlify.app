Implementation Notes

  • Add or edit articles in the articlesData array inside /js/articles.js.
  • Set image to a file in /images/ (or any local path). If left blank, the built-in fallback style appears.
  • Set popular: true on any article you want to appear in the Most Popular section.
  • For article pages, include /js/toc.js and add: <aside class="article-toc" data-toc data-toc-container=".article-content"></aside>
